Open:
Please use the “GitHub.xcworkspace” file to open the project.

Implementation available in this App.
1. The login screen.
2. The profile page for the logged in user.
	a. It has a brief information about the user.
	b. User’s top four repositories based on the star count of the repositories.
	c. A bar graph indicating the contributions of user over the last one year. 
	   The bar graph section header includes the total contributions done in last one year.
3. The app has side bar menu which shows the user name with below options to navigate through the app.
	1. Home
	2. Repositories
	3. Search
	4. Logout
4. Repositories - This view contains the logged in user's repositories sorted based on the last used repository. 
5. Search - This view enables the user to search for the public repositories on the github sorted based on the popularity of 		 the repository.
6. Logout tab on the side menu securely logouts the user from the app.
